# File Cryptography System (Caesar Cipher)

A simple yet effective C-based console application that performs file encryption and decryption using the Caesar Cipher algorithm.

## Features
- **Encryption**: Converts plain text into encrypted ciphertext.
- **Decryption**: Restores encrypted text back to its original form using a secret key.
- **File Handling**: Reads from an input file and saves results to an output file.
- **User Friendly**: Provides a simple menu-driven interface.
